# Volume One Book-Shop-Website
A complete, multi-page book store website that allows users to browse through books, explore their features, and get in touch with the store administration. This project was developed to practice fundamental web development and design skills.
## 🚀 Key Features
* **Home Page:** A welcoming landing page featuring the bookstore's brand logo, a background banner, and a basic form to gather essential visitor information.
* **Products Page:** An attractive grid displaying popular books and world-famous novels, self improvment books, arabic books and kids books with engaging descriptions for each.
* **Contact Us Page:** A structured registration and feedback form for receiving visitor inquiries (including fields for full name, email, gender, phone number, and comments).
* **About Us Page:** An introductory section presenting the development team, university course details (CIS 201), and the group's academic accomplishments.
## 🛠️ Technologies Used
* **HTML5:** Used to build the core structure of all website pages, navigation links, and input forms.
* **CSS:** Used for layout styling, color schemes, typography, and ensuring a neat, user-friendly presentation.
* **JavaScript:** Used to add interactivity, handle form inputs, and enhance overall user experience.
